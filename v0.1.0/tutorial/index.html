Optimization

To demonstrate the package's capabilities, let's apply it to the Rosenbrock function, a well-known benchmark in numerical optimization

Defining a Problem

To optimize a function using HierarchicMemeticStrategy.jl, you must encapsulate your objective function and its constraints within a FunctionProblem object.

First define the Rosenbrock function:

julia> using HierarchicMemeticStrategy
julia> rosenbrock(x) = (1.0 - x[1])^2 + 100.0 * (x[2] - x[1]^2)^2;

Next, wrap it in a FunctionProblem object, specifying the bounds for each variable and whether to maximize or minimize:

julia> lower_bounds = [-5.0, -5.0];
julia> upper_bounds = [5.0, 5.0];
julia> problem = FunctionProblem(
           fitness_function = rosenbrock,
           lower = lower_bounds,
           upper = upper_bounds,
           maximize = false
       );
HierarchicMemeticStrategy.FunctionProblem — Type
FunctionProblem(; fitness_function, lower, upper, maximize=false)

Construct an optimization problem for a standard mathematical function.

Arguments

  • fitness_function::Function: The objective function to evaluate. It must accept a single vector x and return a scalar numerical value.
  • lower::Vector: A vector specifying the minimum allowable values for each dimension.
  • upper::Vector: A vector specifying the maximum allowable values for each dimension.
  • maximize::Bool: (Optional) Set to true for maximization, or false for minimization (default).

Examples

problem = FunctionProblem(
    fitness_function = x -> x[1]^2 + x[2]^2,
    lower = [-5.0, -5.0],
    upper = [5.0, 5.0]
)
source

Performing Optimization

Once Function Problem is defined, we can perform optimization using hms algorithm.

julia> res = hms(optimization_problem=problem)┌────────────────────── HMSResult Summary ───────────────────────

Configuration

The example above uses default configuration. For more configuration details check HMS Configuration section.

Obtaining Results

Once the optimization process is complete, the solver returns an HMSResult object. This object contains the full history of the run, the best found solution, and tools for visualization.
